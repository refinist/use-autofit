# use-autofit

> Vue 3 composable for screen size adaptation with autofit.js wrapper

## Install

Using pnpm, yarn, or npm

```bash
# with pnpm
pnpm add -D use-autofit

# with yarn
yarn add -D use-autofit

# with npm
npm i -D use-autofit
```
